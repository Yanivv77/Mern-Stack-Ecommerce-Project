# Usage

### Env Variables

Change .envExample file to .env 

In File change to your mongodb uri


### Install Dependencies (frontend & backend)

```
npm install
cd frontend
npm install
```

### Run frontend port 3000 and backend port 5000

```

npm run dev

```

### Import data to your database

```
npm run data:import
```
